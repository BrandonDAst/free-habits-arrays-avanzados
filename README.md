# FreeHabits 🎯

Una aplicación web progresiva (PWA) para crear hábitos, registrar check-ins diarios o semanales y visualizar tus rachas para mantener la constancia.

## 🌐 Demo

[https://freehabits.website](https://freehabits.website)

## ✨ Características

- 📝 Crear hasta 5 hábitos personalizados
- 📅 Registrar check-ins diarios o semanales
- 🔥 Visualizar rachas actuales
- 📊 Tablero mensual para seguimiento
- 📱 Instalable como PWA
- 💾 Datos guardados localmente en el navegador

## 🛠️ Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla)
- PWA (Progressive Web App)

## 🚀 Instalación

1. Clona el repositorio:

```bash
git clone https://github.com/gndx/freehabits.website.git
```

2. Abre `index.html` en tu navegador o usa un servidor local:

```bash
npx serve .
```
